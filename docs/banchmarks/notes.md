# Benchmarks — escalas de hashrate

Hashrate (`H/s`) usa os prefixos de KB/MB/GB — cada um é **1000x** o anterior:

| k | M | G | T | P | E |
|---|---|---|---|---|---|
| mil | milhão | bilhão | trilhão | quatrilhão | quintilhão |

## Onde cada coisa se encaixa

| Hardware | Hashrate |
|---|---|
| ESP32 (estimado) | ~50-250 kH/s |
| CPU (i7-10700, 16 threads) | ~9 MH/s |
| GPU dedicada (estimado) | ~1-2 GH/s |
| ASIC S19/S21 (estimado) | ~100-200 TH/s |
| Rede Bitcoin inteira | ~800 EH/s |
